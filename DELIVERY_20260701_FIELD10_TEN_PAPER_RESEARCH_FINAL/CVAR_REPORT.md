# CVaR Tail-Risk Report

The layer calculates empirical 95% VaR, 95% CVaR, tail count, expected adverse excursion, CVaR-to-volatility ratio, A–D tail grade and research risk penalty.

Preferred basis: realized net strategy return or PnL including available costs. Fallback basis: clearly labeled negative completed-H1 market-return proxy. Minimum sample: 60 observations. No precise statistic is published below that threshold.

No live strategy PnL database was available in the container, so no production CVaR value is claimed.
