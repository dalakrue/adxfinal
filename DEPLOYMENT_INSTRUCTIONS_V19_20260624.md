# Deployment Instructions
1. Use Python 3.12.
2. Install: `pip install -r requirements.txt`.
3. Streamlit main entry file: **app.py**.
4. Local start: `streamlit run app.py`.
5. Streamlit Cloud main file path: `app.py`.
6. On first use, open Settings and click **Run Calculation + Open Lunch (One Click)** once.
7. Confirm the published run_id and broker candle time match across Lunch Fields 2–8 and AI Assistant.
