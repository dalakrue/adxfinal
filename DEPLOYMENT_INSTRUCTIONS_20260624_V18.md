# Deployment Instructions

1. Deploy the complete folder unchanged.
2. Select `app.py` as the Streamlit Cloud main file.
3. Use Python 3.12 (`runtime.txt` and `.python-version` are included).
4. Install `requirements.txt`.
5. Configure optional API secrets in `.streamlit/secrets.toml` using the included example.
6. Open Settings and press **Run Calculation + Open Lunch** once to publish the canonical generation and research sidecar.
7. Open **AI Assistant** directly from the main menu.
