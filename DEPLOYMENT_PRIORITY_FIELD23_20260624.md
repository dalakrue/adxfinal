# Streamlit Cloud Deployment

1. Extract the delivered ZIP to a clean repository.
2. Use Python 3.12 as specified by `.python-version`/runtime configuration.
3. Install `requirements.txt`.
4. Copy `.streamlit/secrets.example.toml` to Streamlit Cloud secrets and enter optional credentials.
5. Select the existing main Streamlit file documented by the project README.
6. Deploy, open Settings, and click `Run Calculation + Open Lunch (One Click)`.
7. Open Field 2 and Field 3 priority expanders; ordinary reruns only render the persisted sidecar.
