# Deployment Instructions

Main Streamlit entry file: `app.py`.

1. Use Python 3.12 as specified by the project runtime files.
2. Install dependencies: `pip install -r requirements.txt`.
3. Start locally: `streamlit run app.py`.
4. In the app, open Settings and click **Run Calculation + Open Lunch (One Click)** once.
5. Deploy to Streamlit Cloud with `app.py` as the main file.
