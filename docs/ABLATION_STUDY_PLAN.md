- calibration;
- meta-labeling;
- regime lifecycle;
- multi-scale volatility;
- conformal intervals;
- Bellman control;
- drift detection;
- TFT;

Use identical chronological windows, costs, CPCV paths, and outcome-settlement rules. Report deltas in Brier, ECE, interval coverage/width, expected utility, drawdown, turnover, and action-class frequencies. No component is described as beneficial without stable out-of-sample incremental value.
