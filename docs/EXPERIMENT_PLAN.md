# Experiment Plan

Registered comparisons:

1. uncalibrated versus calibrated probabilities;
2. global versus regime-specific calibration;
3. production direction versus meta-labeled action;
4. fixed versus validated dynamic threshold;
5. fixed interval versus EnbPI;
6. EnbPI versus Bellman-controlled interval;
7. single-scale versus multi-scale volatility;
8. sentiment-only versus event-response memory;
9. each module versus CRCEF-SV;
10. CRCEF-SV with versus without drift protection.

Each experiment records dataset, training, calibration and test windows; CPCV configuration; costs; Brier, ECE, log loss, accuracy, precision, recall and F1; coverage and width; expected utility; maximum drawdown; turnover; five-action rates; PBO; statistical test; confidence interval; and conclusion.

Required baselines include previous close, zero return, historical/rolling mean, logistic regression, calibrated simple classifier, existing production decision, ATR/fixed-width interval, majority class, and simple sentiment.
