# Extreme Mobile Lite Architecture

Mobile Lite Mode is a presentation/retrieval mode. It cannot alter calculations. It uses AUTO/MOBILE LITE/FULL INTERFACE selection, one-shot detection when available, persistent session state and manual override.

In Lite mode the app applies minimal CSS, hides expensive decoration, renders one selected field, limits Table 2 to 10/25 rows and selected column groups, avoids constructing hidden research fields and reads same-candle cached outputs. Full exports remain available through explicit controls.
